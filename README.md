# Haskell Course Notes

Learning Haskell one week at a time from [Introduction To Haskell Programming By Prof. S. P. Suresh | CMI](https://onlinecourses.nptel.ac.in/noc26_cs12/preview).

This repository collects the exercises, experiments, and assignments I have worked through while following the course.

## What I have covered

- Basic function definitions, recursion, guards, pattern matching, and arithmetic helpers.
- Working with lists, list recursion, folds, and simple higher-order functions.
- Introductory `IO` with `main`, `do` notation, `getLine`, `putStrLn`, and simple command loops.
- Custom data types, type instances, and user-defined `Show` implementations.
- Algebraic data structures such as a natural-number type and a binary search tree dictionary.
- Recursive evaluation and parsing of expressions using tokens and postfix notation.
- Common algorithmic exercises such as factorial-style powers, Fibonacci, prime generation, merge sort, and palindrome checks.
- Small interactive programs, including a list manager and a simplified Wordle-style game.

## Weekly Progress

### Week 0

Warm-up exercises and utility functions such as `head`, powers, integer helpers, and simple recursive definitions.

### Week 1

Recursion and numeric problems, including GCD, divisor search, counting digits, reversing numbers, and working with powers of 5.

### Week 2

Lists, list concatenation, custom recursive helpers, trace-style recursion, and additional list transformations.

### Week 3

Introductory `IO` and basic programs that print output from a `main` function.

### Week 6

Custom data types and expression handling: natural numbers, prime generation, tokenisation, postfix parsing, and expression evaluation.

### Week 7

String and list processing, including palindrome detection and longest descending suffix logic.

### Week 8

Merge sort and divide-and-conquer list processing.

### Week 9

An interactive list tool with add, delete, sort, reverse, and quit commands.

### Week 10

A binary search tree dictionary with insert, query, delete, and inorder traversal.

### Week 11

More `IO` and recursion exercises, including boolean aggregation and recursive accumulation.

### Week 12

A simplified Wordle-style game with hint generation and repeated guessing.

## Repository Layout

- `week0(random)/`: extra practice and warm-up functions.
- `week1/` to `week12/`: weekly course work, examples, and assignments.
- `README.md`: course overview and progress summary.
